<?php
// Die Abfrage ausführen und das Ergebnisset abrufen
$sql = "SELECT DATE1 FROM Table1";
$stmt = $conn->query($sql);

// Eine Zeile Daten abrufen
$row = $stmt->fetch(PDO::FETCH_ASSOC);

// Methode 1: Verwenden von echo zur Ausgabe des Werts
echo $row['DATE1'];

// Methode 2: Verwenden von var_dump zur Anzeige von Typ und Wert
// var_dump zeigt sowohl den Typ als auch den Wert der Variablen an, was beim Debuggen hilfreich ist
var_dump($row['DATE1']);
?>

  • Abfrage ausführen:

  • Verwenden Sie $conn->query($sql), um die Abfrage auszuführen und das Ergebnisset zu erhalten. Dann wird $stmt->fetch(PDO::FETCH_ASSOC) verwendet, um eine Zeile der Daten abzurufen und in der Variablen $row zu speichern.

  • Daten ausgeben:

    • echo wird verwendet, um den Wert des Feldes DATE1 auszugeben.
    • var_dump zeigt sowohl den Typ als auch den Wert von DATE1 an, was beim Debuggen nützlich ist.
